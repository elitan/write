# Write

Minimal markdown notes app for macOS.

## Features

- Local markdown files stored in `~/notes`
- Command palette (⌘K)
- Quick note creation (⌘N)
- Autosave

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Stack

Tauri, React, TypeScript, Tailwind CSS, CodeMirror
